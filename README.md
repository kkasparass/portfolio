## Personal portflio website

This is a simple portfolio website to show off my personal skillset and have a more visual overview of my career history and personal project I work on ʕ·͡ᴥ·ʔ


### ✨ Technologies

- **Next.js**
- **React**
- **TypeScript**
- **Tailwind CSS**


### ☁️ Deployment

This portfolio is deployed and hosted on **Vercel** and is available live at [`kasparas.dev`](https://kasparas.dev/).

### 🧪 Running the Project

1. **Clone** the repository.
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Start the development server**:
   ```bash
   npm run dev
   ```
4. **Open** `http://localhost:3000` in your browser.
